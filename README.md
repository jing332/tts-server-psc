# TTS Server PSC

```shell
监听本地文件 test.js 变化并自动推送到 TTS Server:
./tts-server-psc watch -path "./test.js" -addr "192.168.0.106:4566" -ui

[-ui (推送后预览UI) / -debug (推送后debug)]
```
